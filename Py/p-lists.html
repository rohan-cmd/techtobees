Python Lists

List is a collection data type in python. It is ordered and allows duplicate entries as well. Lists in python need not be homogeneous, which means it can contain different data types like integers, strings and other collection data types. It is mutable in nature and allows indexing to access the members in a list

To declare a list, we use the square brackets.

List is like any other array that we declare in other programming languages. Lists in python are often used to implement stacks and queues. The lists are mutable in nature. Therefore, the values can be changed even after a list is declared.

my_list = [16,27,59,77,84,91,96]
Indexing :

To access a value from a list, we use the index values. Below is the code to get the letter ‘D’ from the list containing the letters of the word ‘HELLOWORLD’.

a = ['H','E','L','L','O','W','O','R','L','D']
print(a[9])
print(a[-1])
Output :
D
D

List Operations In Python

Python List Methods

append() - Add an element to the end of the list
extend() - Add all elements of a list to the another list
insert() - Insert an item at the defined index
pop() - Removes and returns an element at the given index
clear() - Removes all items from the list
index() - Returns the index of the first matched item
count() - Returns the count of number of items passed as an argument
sort() - Sort items in a list in ascending order
reverse() - Reverse the order of items in the list
copy() - Returns a shallow copy of the list

Creating a List

Lists in Python can be created by just placing the sequence inside the square brackets[]. Unlike Sets, list doesn’t need a built-in function for creation of list.

# Python program to demonstrate
# Creation of List

# Creating a List
List = []
print("Blank List: ")
print(List)

# Creating a List of numbers
List = [5, 17, 52]
print("\nList of numbers: ")
print(List)

# Creating a List of strings and accessing
# using index
List = ["Python", "JavaScript", "PHP"]
print("\nList Items: ")
print(List[0])
print(List[2])
# Creating a Multi-Dimensional List
# (By Nesting a list inside a List)
List = [['Python', 'JavaScript'] , ['PHP']]
print("\nMulti-Dimensional List: ")
print(List)
Output :
Blank List:
[]

List of numbers:
[5, 17, 52]

List Items:
Python
PHP

Multi-Dimensional List:
[['Python', 'JavaScript'], ['PHP']]
