Python If....else

An if-else statement executes one set of statements when the condition is true and a different set of statements when the condition is false. In this way, an if-else statement allows us to follow two courses of action. The syntax of if-else statement is as follows:

Syntax :
if condition:
# if block
statement 1
statement 2
and so on

else:
# else block
statement 3
statement 4
and so on:

Example : Program to calculate the area and circumference of the circle

radius = int(input("Enter radius: "))

if radius >= 0:
   print("Circumference = ", 2 * 3.14 * radius)
   print("Area = ", 3.14 * radius ** 2 )
else:
   print("Please enter a positive number")
Output :
Enter radius: 5
Circumference = 31.400000000000002
Area = 78.5

Now our program shows a proper response to the user even if the condition is false which is what is wanted.

Nested if and if-else statement

We can also write if or if-else statement inside another if or if-else statement. This can be better explained through some examples:

if statement inside another if statement.

Example : A program to check whether a student is eligible for the loan or not

gre_score = int(input("Enter your GRE score: "))
per_grad = int(input("Enter percent secured in graduation: "))

if per_grad > 70:
   # outer if block
   if gre_score > 150:
      # inner if block
      print("Congratulations you are eligible for loan")
else:
  print("Sorry, you are not eligible for loan")
First Output :
Enter your GRE score: 200
Enter percent secured in graduation: 77
Congratulations you are eligible for loan
Second Output :
Enter your GRE score: 160
Enter percent secured in graduation: 60
Sorry, you are not eligible for loan

This program has one small problem. Run the program again and enter gre_score less than 150 and per_grad greater than 70 like this:

Third Output :
Enter your GRE score: 100
Enter percent secured in graduation: 75

As you can see the program doesn’t output anything. This is because our nested if statement doesn’t have an else clause.
